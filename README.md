## Requirements

- iOS 10.0+
- Xcode 12

## Installation

Follow the first part of this article to get your development environment ready.
https://ludwiguer.medium.com/create-a-react-native-app-with-navigation-between-screens-may-2020-41950226cda9

Clone this repository

```sh
git clone https://github.com/duomenis/friga.git
```

Install the dependencies

```sh
yarn install
```

Install pods

```sh
npx pod-install ios
```

Run the application

```sh
npx react-native run-ios
```
